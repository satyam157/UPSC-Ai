## 🚀 Quick Start (3 Steps)

### Step 1: Take a Few Quizzes
```
→ Open UPSC AI System
→ Go to CA Quiz / PDF Quiz / Practice
→ Complete 5-10 quizzes (different types)
→ This gives Ask Esu meaningful data to analyze
```

### Step 2: Open Ask Esu
```
→ Click "Ask Esu" in sidebar
→ You'll see welcome screen explaining features
→ Ready to generate your first plan!
```

### Step 3: Generate Study Plan
```
→ Enter your study challenge/goal
  (e.g., "I'm weak in CA, need to improve")
→ (Optional) Select exam date
→ Click "🚀 Generate Personalized Study Plan"
→ Wait 1-2 minutes for AI processing
→ View results in 4 tabs
→ Export or generate new plan
```

## 🔧 Troubleshooting

### Ask Esu doesn't appear in sidebar
- **Solution:** Restart Streamlit app
- **Check:** Is app.py saved with updates?

### "No quiz data available" message
- **Solution:** Take at least 3-5 quizzes first
- **Action:** Complete quizzes, then return to Ask Esu

### Study plan seems generic
- **Solution:** Provide more specific prompt
- **Example:** Instead of "help me study"
  → "I score 50% in CA, need 75% in 2 months"

### AI response takes very long
- **Normal:** 1-2 minutes is expected
- **Groq API:** Can sometimes have delays
- **Solution:** Try again in a few minutes

### Export files are empty
- **Solution:** Wait for generation to complete
- **Check:** Ensure no red error messages
- **Fix:** Try generating again
